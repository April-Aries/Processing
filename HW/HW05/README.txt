注意事項
1. 請不要刪除或修改 Rank.txt 之文件，若刪除則會讓程式跑不動，若修改則會讓排行榜不同步，詳細解釋請看 Rank 按鈕解釋
2. 為了避免按鈕連續點擊 ( 因為 frameRate 太高讓按鈕重複讀取 )，我將 frameRate 調成每秒更新十次，因此使用起來會有些卡，再麻煩助教包含
3. 文字輸入框目前很不完善，請助教詳細閱讀 Import 按鈕解釋再測試程式會比較適合

由於此次作業我想將其與專題結合，製作的較為複雜，所以提供一份說明文件幫助助教了解程式架構。

這份作業包含了六個 .pde 檔，因為不同頁面我將其放在不同檔案方便我撰寫程式碼，這邊稍微介紹每個 .pde 檔的用途
1. HW05_WebAccess.pde: 程式主架構，包含 setup() 及 draw() 兩支函式
2. class.pde: 按鈕 class
3. Home.pde: 主頁面
4. Import_Score.pde: 輸入分數頁面
5. TEXTBOX.pde: 輸入文字框 class
6. View_Rank.pde: 顯示排行榜頁面

當開始執行程式，會進入遊戲主畫面，可以看到畫面中有四個按鈕，以下為按下後會做的事情：
1. Rank: 跳轉至排行榜頁面
2. Import: 跳轉至輸入成績頁面
3. Read: 讀取雲端 CSV 檔資料 ( 雲端排行榜 )
4. Exit: 離開程式，程式會直接關掉

Read 按鈕按下去後會讀取雲端 CSV 檔資料，並將其印在 console 中，請去 console 查看資料是否正常匯入，理論上在助教未進行任何輸入成績時，應該可以讀入幾筆我是先輸入的資料，因此助教測試輸入的成績及姓名並不會從第一筆開始

Import 按鈕按下去後會跳轉至輸入成績頁面，頁面雖然寫 Game Over 但請不要緊張，這只是使用專題之背景而已，請助教在文字輸入框 ( 灰色處 ) 輸入欲填入之姓名及分數。在文字框使用上，請先使用滑鼠點擊一下，可以看到灰色輸入框顏色會變淡，代表以選擇到該輸入框，此時再輸入數據即可，若想取消輸入，請跳出此頁面再進入即可。若想跳離此頁面，可以點選下方 HOME 按鈕，即可回到遊戲主畫面，若要送出資料，按下 Send 按鈕即可，若有成功送出會在底下看到一行紅字告訴資料已送出，若有文字輸入框留空，則會有警告訊息跳出，並不會送出資料。
但有幾點須特別注意：
1. 文字框目前支援英文大小寫、數字，但不支援任何特殊符號以及中文字，如果助教的輸入法維持在中文，可能會無法輸入
2. 文字輸入框會將姓名當成字串、分數當成整數，請助教不要嘗試在分數文字輸入框輸入非數字之字元，否則會有不可預期之錯誤發生！
3. 文字輸入框尚未設計完善，所以十分脆弱，目前遇到的問題如下：可能會一次輸入多個字元或按下鍵盤無輸入產生，屆時再請助教慢慢按鍵盤，如果沒有顯示輸入結果，請再多按一次，我知道這樣很難用，但目前還在改善它，敬請期待期末專題 demo 的成果。
4. 按鈕點擊上，如果點太快或按著不放，可能會因為感應太快導致程式判斷你按下下個頁面同位置之按鈕，若不慎發生此問題請重新執行程式。

Rank 按鈕按下後會跳轉至排行榜頁面，這一頁只會顯示排行榜中前五名的資料，不過這邊讀取是使用單機版排行榜讀取，不是使用雲端資料，但繳交作業前我已讓兩邊資料同步，理論上兩邊寫入之資料應該是一樣的
也是因為這個原因，請不要動到資料夾中 Rank.txt 文件，不然程式會跑不起來！
至於為甚麼不用雲端 CSV 檔讀取：因為 loadTable 函式無法讀取我這份雲端 CSV 檔，因為結尾並不是 .csv 所以不能使用，如果要使用只能用 loadStrings() 再進行 split()，但要在實作有點複雜，所以這次作業先不實作出來，一樣在期末專題 demo 時會將其功能補上！
