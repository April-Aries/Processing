HW04資料夾為此次作業之主要函式，請助教直接看此資料夾內容評分。
Create_Star_BG 資料夾為生成背景圖片之函式，這支函式是為了方便我們快速生成背景圖片而製作，產生支背景圖片會直接拉進 HW04 資料夾中 data 資料夾使用
README 文件則是方便助教了解此次作業繳交以及相關作業說明

此次作業我們設定的場景為遨遊在宇宙中的飛船，這是一款全螢幕的場景，執行程式後螢幕會被程式佔據，若需離開只需要按 ESC 鍵即可。

接下來將進行程式說明

背景我們使用拼貼方式實作，我們利用 Create_Star_BG 資料夾中 Create_Star_BG.pde 的程式產生十張背景圖片，並隨機貼上，其中每 1/3 秒會更新一次背景，透過這種方式產生星點閃爍的特效

關於飛船的移動，我們使用滑鼠操控，在以畫面中心點正負 200 pixel 的範圍以外才可以成功移動，否則飛船會停留在原地，跟隨滑鼠方向轉動，且會看到飛船後方的火苗會消失，代表飛船沒有在移動。當飛船可以移動時，飛船會跟著滑鼠方向以等速率移動，且飛船後方的火苗會出現，代表飛船有在移動

場景上的行星設有六顆，有一半在飛船前方，所以當飛船經過時會被行星阻擋，另外一半在飛船後方，所以當飛船經過時會蓋過行星。

請注意，飛船可以無限制往邊界移動，並不會出現問題，但當往四周超過 0.5 個 pixelWidth 或 PixelHeight 時，就不會有行星出現。

若助教在執行有任何問題，歡迎與我們聯繫！
